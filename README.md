# Smart Construction Management System

## 📌 Overview
This project is a MySQL-based system to manage construction projects, workers, materials, and cost tracking.

## 🚀 Features
- Project tracking
- Worker allocation
- Material usage
- Cost calculation using SQL

## 🧠 SQL Concepts Used
- Joins
- Aggregation
- Foreign Keys

## 📂 Files
- schema.sql → database structure
- data.sql → sample data
- queries.sql → analysis queries
